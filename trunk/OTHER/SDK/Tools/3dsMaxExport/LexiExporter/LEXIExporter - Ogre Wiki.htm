[edit]

Concept

LEXIExporter is a new native plugin for 3DSMax8. It is a project in progress, with realeases every 2. or 3. week, and with your help it will be just what you need.

The concept behind the LEXIExporter consists basically of two ideas. First of all, it should be extendable. So far this means opensource and plugable. It will be possible to create plugin export types, such a specific scene formats. It will also be possible to create different methods of material exports, eg. extend the standard emulation to more optimized shaders or export materials enforcing a certain parent-child relation. Secondly, loose coupling to the modelling tool (so far it´s 3DSMax8) should be kept where possible.

[edit]

NOTICE

Please note that LEXIView is currently compatible with the Dagon (1.2.x) branch only. Meshes exported from LEXIExporter are also in Dagon format but if you use Eihort they will be upgraded automatically on loading.

[edit]

Features

LEXIExport currently features:

  • Static Mesh export
    • Vertex Colors
    • Normals
    • Multiple UV sets
    • Collaps Hierarchy into single geometry
    • Reindexing optimization
  • Skinned Mesh export
    • Vertex Colors
    • Normals
    • Multiple UV sets
    • Multiple Animations
    • Multiple root bones
    • Linked bones
  • Shader based materials, to mimic 3DMax DX Viewport look (so far only one at a time)
    • Blinn colors
    • Diffuse texture
    • Specular Color
    • Specular Level
    • Self-Illumination
    • Submesh materials
  • Pose Support (on both skinned and static mesh)
    • Single Pose Data
    • Sampled Pose Animation
  • Add Selection of Multiple Objects

For more details see the Changlog


Design & Features to come:

  • Templated settings.
  • Morph animation export
  • Plugin based exporter types
    • i.e. Scenemanager specific scene exports
  • Plugin based material export.
    • i.e. override standard material export of Max materials with Diffuse+Specular+Bump maps assigned
    • i.e. add support for new material types

How to Use

Take a look at the screenshot descriptions and the video tutorials. If more questions arise post them in the LEXIExporter thread on the ogreforum. Feature requests, Ideas or bug reports may be reported as a private message to me(sethos) in the ogreforum.

Screenshots

The LEXIExporter is a native plugin for 3dMax8, and may be positioned in the Max Utility Panel as shown
Clicking the Properties button reveals the Export Tree Dialog shown above. Here the user may add objects to be exported and set individual export properties as needed.
When setting the individual properties for an object to be exported, select the object in the tree and set the options in the panel to the right.
During the export a progress window will appear, and afterwards a log window will popup if any serious warnings or errors have occoured. The log window may also be accessed through the properties dialog.
